Installation
============

Using pip install:

.. code-block:: console

    pip install -q "git+https://github.com/willem0boone/resample_zarr#egg=resampling"

Install a specific version (e.g. v1.0.0)

.. code-block:: console

    pip install -q "git+https://github.com/willem0boone/resample_zarr@v1.0.0#egg=resampling"
